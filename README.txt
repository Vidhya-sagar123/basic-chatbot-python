Project Name: Basic Chatbot

Description:
This project is a simple rule-based chatbot developed using Python.
The chatbot interacts with the user by responding to predefined inputs.

Features:
- Responds to greetings
- Answers simple questions
- Ends conversation when the user types "bye"
- Simple and user-friendly interface

Concepts Used:
- Functions
- While Loop
- If-Elif-Else Statements
- Input/Output Handling

Files Included:
1. chatbot.py - Main Python program
2. README.txt - Project documentation

How to Run:
1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the command:

python chatbot.py

Sample Inputs:
hello
how are you
what is your name
bye
